About This App

This web application helps you find historical crime data in Bristol. Use the tools below to explore crime statistics in different neighborhoods.

This web application helps you find historical crime data in Bristol. It uses dataset from Bristol Open Data website by UK Government to provide accurate and up-to-date information about crimes in different neighborhoods.

Our mission is to make crime data accessible and easy to understand for everyone.
